# Razonamiento Inductivo y Deductivo
Estos 2 metodos de razonamientos sirven para realizar una [investigacion cientifica](obsidian://open?vault=Neuroscience&file=Ciencia).

Sirven para llegar a una conclusion. En donde difieren estos 2 tipos de razonamiento es en *la forma* y *los pasos* para obetener esa conclusion.

Dependiendo del problema puedes utilizar uno u otro metodo. En lo primero que se debe fijar uno es en las premisas del problema y distinguir si empiezas de lo particular o de lo lo general.

## Razonamiento Inductivo

### Caracteristicas
- Las premisas son particulares
- Las conclusiones son generales
- Se utiliza generalmente para obtener conocimiento nuevo
- Se debe tener cuidado ya que al iniciar con premisas particulares, se pierde mucha informacion sobre lo general (universo) del conjunto

## Razonamiento Deductivo

### Caracteristicas
- Las premisas son generales
- Las conclusiones son particulares
- Se utiliza cuando ya se tiene un conocimiento previo sobre el tema y se trata de llegar a una conclusion
